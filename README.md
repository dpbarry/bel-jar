# <img src="assets/logo.svg" alt="BelJar Logo" width="42" height="42" align="top"> BelJar

**BelJar** is a modern in-browser IDE for the [Beluga](https://github.com/Beluga-lang/Beluga) language. It uses `js_of_ocaml` to adapt the Beluga compiler into Javasscript, which allows it to run entirely client-side.

It's currently in its beginning stages. In the future, it can be a staging ground for experimental Beluga features.

Live at [dpbarry.github.io/bel-jar/](https://dpbarry.github.io/bel-jar/)
